# APG-React

An implementation of the [Patterns](https://www.w3.org/WAI/ARIA/apg/patterns/) in the ARIA Practices Guide as React Components.

Although we have tried to faithfully implement these patterns, including using [APG-Gherkin](https://github.com/AFixt/apg-gherkin/), some requirements still must be satisfied by the implementers. This is especially true of things like ensuring proper contrast, creating accessible labels, and dictating focus for when dialogs are closed. 

Like any project, there are bound to be bugs or things that could be done better. [Contributions](./CONTRIBUTING.md) are welcome and encouraged.
